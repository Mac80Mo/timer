# Desktop Timer

A minimal countdown timer for the desktop, built with Electron and TypeScript.

## Features

- Countdown timer with audio alarm on completion
- Input formats: `MM:SS`, `5m 30s`, or plain minutes
- Mute toggle, light/dark theme
- Runs on Linux, macOS, and Windows

## Requirements

- Node.js >= 22

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run package
```

Produces a distributable in the `release/` directory.

## Tests

```bash
npm test
```

# Desktop Timer (Deutsch)

Ein schlanker Countdown-Timer fur den Desktop, entwickelt mit Electron und TypeScript.
